# openbox-themes
  - Themes I have made for the Openbox Window Manager

# Installation
  - extract or clone openbox-themes

  - cd into openbox-themes

  - sudo cp -r * /usr/share/themes/

  - You can slecet them in obconf or by editing your rc.xml
